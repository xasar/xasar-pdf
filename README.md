# 📄 Browser based PDF Modifer

A powerful, 100% client-side, browser-based PDF utility built with modern web technologies. This tool specializes in document layout manipulation, media insertion, and visual markup directly inside the browser—with zero server uploads and complete data privacy.

> ⚠️ **Note on Editing:** This tool is a visual layout modifier and builder. It allows you to append, rearrange, and overlay new elements (text, images, shapes). It **does not** edit or rewrite existing structural PDF text.


🚀 **Live Demo:** [xasar.github.io/pdf](https://github.io)
---

## ✨ Features

### 🧩 Page Manipulation
* **Drag-and-Drop Reordering:** Rearrange page sequences visually.
* **Page Deletion:** Remove unnecessary pages with a single click.
* **Document Merging:** Combine multiple PDF files into one cohesive document.
* **Page Rotation:** Fix orientation issues instantly (90° / 180° / 270°).

### ✍️ Visual Overlays & Markup
* **Rich Text Overlays:** Place new text blocks anywhere on the page with adjustable sizing.
* **Image Insertion:** Upload and position PNG/JPEG images or logos onto your document.
* **Special Characters & Symbols:** Easily inject mathematical, currency, or custom symbols.
* **Freehand Drawing:** Annotate, sign, or sketch directly on top of the canvas.

### 🔒 Privacy & Performance
* **100% Client-Side:** All processing happens locally in your browser.
* **Absolute Data Privacy:** Your sensitive documents are never uploaded to an external server.
* **Instant Exports:** Download your newly compiled PDF immediately with zero rendering delay.

---

## 🛠️ Built With

* **Frontend Framework:** [HTML/CSS/JavaScript]
* **PDF Rendering Engine:** `pdfjs-dist` (Mozilla PDF.js)
* **PDF Modification Library:** `pdf-lib`
